#### 向量的点乘与几何意义

1. 两个向量“相乘”，结果是一个数！（标量）
2. 两个向量相乘，更严格的说法：两个向量的点乘，或两个向量的内积
3. 向量点乘的应用
   - 判断两个向量的夹角。
      - 如果点乘结果 = 0 ,两个向量垂直
      - 如果点乘结果 > 0 ,两个向量夹角为锐角
      - 如果点乘结果 < 0 ,两个向量夹角为钝角
   - 判断两个向量的相似程度（推荐系统）
   - 几何计算中，求投影点的坐标

### 矩阵 Matrix

#### 矩阵的基本运算

1. A + B = B + A
2. (A + B) + C = A + (B + C)
3. (ck)A = c(kA)
4. (c+k)A = cA+kA
5. k(A+B) = kA + kB
6. 存在矩阵O,满足：A + O = A
7. 存在矩阵-A,满足：A +（-A）= 0 ，且 -A 唯一
